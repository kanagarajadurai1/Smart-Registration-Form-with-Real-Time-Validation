# 🚀 Smart Registration Form with Real-Time Validation

## 📌 Project Overview
The **Smart Registration Form** is a modern web application built using **HTML, CSS, and JavaScript** that performs **real-time form validation**. The system ensures that users provide valid information before submitting the form, improving both **data accuracy** and **user experience**.

---

## ✨ Features

🔹 Real-time form validation  
🔹 Name field validation (cannot be empty)  
🔹 Email format verification  
🔹 Password minimum length validation  
🔹 Password strength indicator  
🔹 Show / Hide password functionality  
🔹 Dynamic error message display  
🔹 Disabled submit button until all fields are valid  
🔹 Responsive and modern UI design  
🔹 Success alert after form submission  

---

## 🛠️ Technologies Used

- **HTML5** – Web page structure  
- **CSS3** – Styling and layout design  
- **JavaScript (ES6)** – Form validation and dynamic functionality

## ⚙️ How It Works

1️⃣ The user enters **Name, Email, and Password**.  
2️⃣ JavaScript listens for **input events in real time**.  
3️⃣ Each field is validated instantly:  
   - Name must not be empty  
   - Email must follow a valid email format  
   - Password must contain at least **6 characters**  
4️⃣ The **Submit button remains disabled** until all inputs are valid.  
5️⃣ Once validated, the user can successfully submit the form.
